## Parseable Server Logs

**File:** `parseable-server-logs-sql.json`

Analyze Parseable server logs, error patterns, request activity, status distribution, and runtime log trends. Use it to troubleshoot Parseable server behavior from runtime log data.

**Source:** `SQL`

**Tags:** `server-logs`, `parseable`, `logs`

**Filter variables:** `dataset`

**Panels:**

- Total Logs
- Error Logs
- Warning Logs
- Error Rate
- Log Volume Over Time
- Error / Warning Trend
- Logs by Level
- Top Modules
- Top Error Modules
- Repeated Error Messages
- Logs by Deployment
- Logs by Customer
- Module Severity Matrix
- Recent Error Logs
